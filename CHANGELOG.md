# Changelog

## v2.2.0 — 2026-05-27

### 新增

- **Auto.cfg 可视化编辑器**：新增专用页面，按分组（鼠标、准星、视频、键位、快速买枪、基础设置、声音、HUD）可视化编辑全局 cfg 目录下的 auto.cfg。
  - 开关控件（0/1 切换）
  - 滑块 + 数字输入（带范围的数值参数）
  - 文本输入（字符串参数）
  - 每条设置右侧显示中文说明
  - 支持从内置模板一键创建 auto.cfg
  - 支持"原始视图"查看完整 cfg 文本
  - Ctrl+S 快捷保存
- **自动更新检测**：启动时通过 GitHub Releases API 检测最新版本，发现更新后弹出横幅提示，用户确认后自动下载安装包并静默重启安装。
  - 启动 3 秒后首次检查，之后每 30 分钟自动检查
  - 显示更新日志和发布时间
  - 下载进度条实时显示
  - 安装完成后自动重启

### 修复

- 修复 CFG 编辑器"保存并关闭"按钮在保存失败时仍会关闭编辑器、导致未保存修改丢失的问题。
- 提取 `Icon`、`CsgoIcon`、`UserIcon` 为共享组件，消除 7 个文件中的重复定义。

### 优化

- 仪表盘统计卡片新增"总大小"，展示所有备份的合计占用空间。
- `list_backups` 引入配置目标缓存（5 秒 TTL），减少对 Windows 注册表和文件系统的重复扫描，页面加载和刷新更快。
- 点击"重新检测"会主动刷新缓存，保证数据一致性。
