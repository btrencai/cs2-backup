# Changelog

## v2.1.1 — 2026-05-27

### 修复

- 修复 CFG 编辑器"保存并关闭"按钮在保存失败时仍会关闭编辑器、导致未保存修改丢失的问题。
- 提取 `Icon`、`CsgoIcon`、`UserIcon` 为共享组件，消除 7 个文件中的重复定义。

### 优化

- 仪表盘统计卡片新增"总大小"，展示所有备份的合计占用空间。
- `list_backups` 引入配置目标缓存（5 秒 TTL），减少对 Windows 注册表和文件系统的重复扫描，页面加载和刷新更快。
- 点击"重新检测"会主动刷新缓存，保证数据一致性。

## v2.1.0 — 2026-05-26

- 首次启动备份目录引导流程。
- 支持中文 WiX `.msi` 安装包。
- 备份条目展示 Steam 用户头像。
- 版权信息展示。

## v2.0.0 — 2026-05-25

- 全局目录与用户目录分类管理。
- 备份管理页面重构（长条式布局、搜索、排序）。
- ZIP 导出功能。
- 多 Steam 账号自动检测与头像展示。
- CFG 编辑器（查看/编辑备份内的 cfg 文件）。
